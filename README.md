# Ngevent-backend
Backend for Ngevent apps

## Build With
1. Typescript
2. Express
4. TypeORM
5. PostgreSQL

## Setup
1. Clone this repository
2. Duplicate **.env.example** file to **.env** on code editor and change the variable
3. Install node_modules with 
``yarn install``
or 
``npm install``
5. Run dev server with
``yarn dev``
or 
``npm run dev``
